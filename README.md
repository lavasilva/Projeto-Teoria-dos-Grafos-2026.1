# Projeto-Teoria-dos-Grafos-2026.1

### Regras de custo de nó:

- Conexões da cidade central de uma região com cidades da mesma região custam **1**.

- Conexões entre duas cidades da mesma região (com nenhuma delas sendo o centro da região) custam **1.5**.

- Conexões entre duas cidades centrais (duas regiões) custam **1.5**.

- Conexões entre uma cidade central de uma região com cidades fora daquela região custam **2**.

- Conexões entre duas cidades de regiões diferentes (com nenhuma delas sendo o centro da região) custam **3**.

As cidades que são centros são:

- Nordeste: REC

- Sudeste: GRU

- Sul: POA

- Centro-Oeste: BSB

- Norte: MAO
